A CLI that unifies C/C++ and Rust security tooling - Cppcheck, Flawfinder, Semgrep, cargo-audit, cargo-geiger and Clippy - under a single command, normalising their heterogeneous outputs into one SARIF 2.1.0 report for GitHub Code Scanning.
